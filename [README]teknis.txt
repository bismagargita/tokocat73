1. TOP
Halaman Utama TOP
- Kursor fokus utama langsung ke TextField Cari
- TextField Cari Enter --> Kursor pindah ke Tabel (Select Baris di Tabel) --> Perpindahan select baris di Tabel menggunakan tombol v/^
Warning !! : Jangan lakukan, TextField Cari Enter --> Kursor pindah ke Tabel (Select Baris di Tabel) --> Enter <Menimbulkan Error>
- Salah satu baris di Tabel di Enter --> Muncul halaman Edit
- Klik salah satu baris --> Klik Tambah/Edit --> muncul halaman Tambah/Edit

New/Edit
- Perpindahan TextField / inputan ke Simpan menggunakan Enter dan di akhiran dengan menyimpan data menggunakan Enter/Klik Button

Note :
-  Sudah refresh otomatis
-  Semua keyEvent done seperti ESC untuk tutup halaman
